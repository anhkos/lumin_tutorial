
# LUMIN v2: Neurosymbolic Query Engine for NASA PDS

This repository contains a series of tutorial notebooks to introduce students on the LUMIN project on LLM and NLP concepts to get them onboarded with the task. 


## Setup Instructions

### 1. Clone the Repository
Open your terminal and run the following commands to download the codebase.

```bash
git clone https://github.com/anhkos/lumin_tutorial.git
cd lumin_tutorial 

```

### 2. Set Up Virtual Environment

Always run this project inside a virtual environment to manage dependencies.

**For macOS / Linux:**

```bash
python3 -m venv venv
source venv/bin/activate

```

**For Windows (PowerShell):**

```bash
python -m venv venv
.\venv\Scripts\Activate.ps1

```

### 3. Install Dependencies

Install the required Python packages.

```bash
pip install -r requirements.txt

```
